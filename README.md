# nanoc-mustache

This provides an [Mustache](http://www.methods.co.nz/mustache/) filter for [nanoc](http://nanoc.ws).

## Installation

`gem install nanoc-mustache`

## Usage

```ruby
filter :mustache
```

This filter takes no options.
